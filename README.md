# 🚀 Next.js Blog Application

A modern, fast, and SEO-friendly blogging platform built entirely with **Next.js** (App Router), TypeScript, and Tailwind CSS. This repository contains the complete source code for the frontend and backend integration of the application.

---

## 🌟 Features

* **⚡ Next.js App Router:** Utilizing Server Components (RSC) for optimal performance and minimal client-side JavaScript.
* **🎨 Responsive Design:** Crafted with Tailwind CSS, featuring full dark mode support and mobile-first layouts.
* **✍️ Rich Text Markdown Editor:** Seamless blog creation supporting Markdown syntax, code highlighting, and live previews.
* **🔍 SEO Optimized:** Dynamic metadata generation, Open Graph (OG) tags, and automatically generated sitemaps.
* **🔐 Authentication:** Secure user login, registration, and session management (e.g., NextAuth.js / Clerk).
* **💬 Interactive Comments:** Built-in comment system with nested replies for user engagement.
* **📈 Analytics:** Integrated page view counters and reading time estimators for each article.
